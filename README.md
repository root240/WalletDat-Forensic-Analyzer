# WalletDat-Forensic-Analyzer

Author: Reza Niruomand

Academic & forensic analysis tool for Bitcoin Core `wallet.dat` files.

## Features
- Wallet.dat structural inspection
- Berkeley DB header analysis
- AES-256-CBC structure detection
- Entropy & randomness profiling
- Corruption & bit-flip detection
- Cryptographic attack feasibility analysis (theoretical)

❌ No brute-force  
❌ No password recovery  
❌ No exploitation  

## Usage
```bash
python3 main.py wallet.dat
